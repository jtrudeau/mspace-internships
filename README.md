# makerSPACE Internships Hub

This repo powers the Dawson makerSPACE internship hub. For interns, this is the main reference for what the program is, where to start, and how to keep moving each week.

## Start Here

1. Open the student launch guide.
   Path: `/mspace-internships/sessions/internship-launch/`
2. Open your track page and work from the current task list.
3. Post your weekly check-in in Discord.

## Track Pages

- Astro Pi Build: `/mspace-internships/sessions/tracks/astro-pi/`
- Pico2 Navigation Robots: `/mspace-internships/sessions/tracks/pico2/`
- Scratch Helpers: `/mspace-internships/sessions/tracks/scratch/`
- LED Making: `/mspace-internships/sessions/tracks/led/`
- RPi Arcade Consoles: `/mspace-internships/sessions/tracks/arcade/`

## What This Hub Covers

- Internship overview and timeline
- Launch guide and expectations
- Per-track task pages
- Publication targets for ScienceFest, SALTISE, Robotics for All, and Dunrae

## Weekly Rhythm

- Check your track page for the current build push.
- Build something small, test it, and document what changed.
- Post one weekly check-in in Discord:
  - what you worked on
  - one thing you made or figured out
  - where you are stuck or need help

## For Maintainers

Run locally:

```bash
npm install
npm run dev
```

Open `http://localhost:3000/mspace-internships/`.

Static build and deploy:

```bash
npm run build:pages
npm run deploy
```
