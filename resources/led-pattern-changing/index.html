LED Pattern Changing

A beginner button-and-LED project: press once to start, then each press cycles through seven different light patterns on six LEDs — pulled from the LED curriculum for a stable internship reference.

Back to ResourcesBack to LED Track

Overview

Prerequisites

  • Arduino Uno
  • LEDs / Switch / Resistors
  • If-Else Statements
  • Programming Basics

List of components

  • 1× Arduino Uno
  • 1× USB Cable
  • 6× LED (any color)
  • 5× 220Ω Resistor
  • 1× 10kΩ Resistor
  • 8× Input M-M wires (green recommended)
  • 2× Red wire
  • 6× Black wire

How does it work?

Press the button once to start pattern 1. Each subsequent press advances through seven patterns. After pattern 7, it resets to pattern 1.

Set Up

LEDs on pins 12, 11, 10, 7, 6, 5. Button on pin 3. Recreate the breadboard layout from the diagram.

Coding

Part 1 — Setup: pin constants (L1–L6, buttonPin), delay de, counter p, buttonState, and void setup() with all pinMode calls.

/*
 * LED Pattern Changing
 * 
 * Fun LED Pattern Changing project on 4 LEDS and 1 button.
 * 
 * The circuit:
 * > LEDS - DI #5, DI #6, DI #7, DI #10, DI #11, DI #12
 * > Switch - DI #3
 * 
 * Created 03/11/2022
 * By Anh Nam
*/

// Led Inputs

const int L1 = 12;
const int L2 = 11;
const int L3 = 10;
const int L4 = 7;
const int L5 = 6;
const int L6 = 5;

const int buttonPin = 3;  // Button input

const int de=50;  // delay time

const int p=0;    // variable for pattern

const int buttonState = 0;    // variable for reading the pushbutton status

void setup() {


    pinMode(L1, OUTPUT);
    pinMode(L2, OUTPUT);
    pinMode(L3, OUTPUT);
    pinMode(L4, OUTPUT);
    pinMode(L5, OUTPUT);
    pinMode(L6, OUTPUT);

    pinMode(buttonPin, INPUT);

}

Bug hunt for interns

The curriculum declares p and buttonState as const int but reassigns them in the loop — they should be plain int. Good exercise to spot and fix.

Part 2 — The loop: read the button, increment p on press, then branch into sequence functions 1–7.

void loop() {
    buttonState = digitalRead(buttonPin);
    if (buttonState == HIGH)
    {
        p++;
        delay(2000);
    }

    if (p == 1) {
           first_sequence();
    }
    else if  (p == 2) {
          second_sequence();
    }
     else if(p == 3) {
          third_sequence();
    }
     else if(p == 4) {
          fourth_sequence();
    }
     else if(p == 5) {
          fifth_sequence();
    }
     else if(p == 6) {
          sixth_sequence();
    }
     else if(p == 7) {
          seventh_sequence();
     }
}

Part 3 — The sequences: seven functions (first_sequence through seventh_sequence) using digitalWrite; seventh_sequence clears LEDs and resets p to 0.
