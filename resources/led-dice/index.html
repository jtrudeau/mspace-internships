LED Dice

Roll a virtual 6-sided die: press the button, watch the animation, then see the dice face on the 8×8 NeoPixel matrix.

Back to ResourcesBack to LED Track

Overview

Prerequisites

  • Arduino Uno
  • LEDs / Switches / Resistors
  • Pseudo Random Number Generation
  • Programming Basics
  • FastLED Library

List of components

  • Arduino Uno Board
  • 5 M-M wires + 3 F-M wires
  • 1× Push Button
  • 1× 220Ω Resistor
  • 1× NeoPixel 8×8 LED Matrix

How does it work?

On button press, a short rolling animation runs (every 10th LED lights green), then a random value 1–6 selects a dice face drawn in green on the matrix. The face clears after a few seconds.

Set Up

Connect DIN → DI #3, +5V → 5V, GND → GND (labels on the back of the matrix). The Fritzing part may differ from the kit matrix — match your hardware.

setColor(x, y) uses 1-based coordinates

The helper subtracts 1 when indexing into the LED array, so x and y start at 1 for the visible grid, not 0.

Animation delay

The curriculum comment says "1 ms" delay between animation LEDs; the sketch uses delay(100) so the roll is visible.

Coding

Part 1 — FastLED setup: grid size, setColor, setup, and a quick diagonal test pattern.

#include <FastLED.h>

// Constant Variables for Rows and Columns
const int NUM_ROWS = 8;
const int NUM_COLS = 8;

const int NUM_LEDS = NUM_ROWS*NUM_COLS;

// Array to hold color data for each LED in the Matrix
CRGB leds[NUM_LEDS];

// Digital Input Pin (DIN)
const int DATA_PIN = 3;

// Pin connecting the Push Button to the Arduino Board
const int BTN_PIN = 5;

void setColor(int x, int y, CRGB color){
  int index = (NUM_ROWS * (y-1)) + x-1;
  leds[index] = color;
}

void setup() {
  FastLED.addLeds<NEOPIXEL, DATA_PIN>(leds, NUM_LEDS);
}

void loop() {
  for(int i = 1; i <= NUM_ROWS; i++){
    setColor(i,i, CRGB(0,0,255));
    FastLED.show();
  }
  delay(10000);

  for(int i = 1; i <= NUM_ROWS; i++){
    setColor(i,i, CRGB(0,0,0));
    FastLED.show();
  }
}

Part 2 — buttonPressed() edge detection and a loop() that reacts to presses.

int buttonPressed(uint8_t button) {
  static uint16_t lastStates = 0;
  uint8_t state = digitalRead(button);
  if (state != ((lastStates >> button) & 1)) {
    lastStates ^= 1 << button;
    return state == HIGH;
  }
  return false;
}

void loop() {
  if(buttonPressed(BTN_PIN)){
    for(int i = 1; i <= NUM_ROWS; i++){
      setColor(i,i, CRGB(0,0,255));
      FastLED.show();
    }
    delay(3000);
  }

  for(int i = 1; i <= NUM_ROWS; i++){
    setColor(i,i, CRGB(0,0,0));
    FastLED.show();
  }
}
